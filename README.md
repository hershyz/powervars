<h3>Object oriented, algorithm framework written in Java used for dealing with variables.</h3>
<br>
<h4>Example:</h4>
<pre>
  powerstring x = new powerstring("abc");
  System.out.println(x.raw);
  System.out.println(x.reverse());
</pre>
<br>
<h4>Output Would Be:</h4>
<pre>
  abc
  cba
</pre>
